# 模块介绍

### [@libmedia/common](https://zhaohappy.github.io/libmedia/docs/index.html)

一些通用的工具方法

:::code-group

```bash [npm]
npm install @libmedia/common
```

```bash [pnpm]
pnpm add @libmedia/common
```

```bash [yarn]
yarn add @libmedia/common
```

:::
