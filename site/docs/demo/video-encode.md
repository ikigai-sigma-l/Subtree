# 视频编码

## wasm 编码器

点击 ```开始``` 开始执行编码操作，点击 ```停止``` 结束编码。点击 ```开始``` 之前可以选择本地文件。没有选择文件会使用测试文件。

代码会加载编码器，编码器托管在 github 上，请耐心等待。
