# 视频解码

## Wasm 解码器

点击 ```开始``` 开始执行解码操作，点击 ```停止``` 结束解码。点击 ```开始``` 之前可以选择本地文件。没有选择文件会使用测试文件。

代码会加载解码器，解码器托管在 github 上，请耐心等待。
